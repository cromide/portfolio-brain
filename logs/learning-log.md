# Learning Log

> 이 파일은 Claude와 연동됩니다. 학습 기록을 추가하면 뉴런 그래프가 업데이트됩니다.
> 사용법: Claude에게 `/brain log` 또는 `/brain update` 프롬프트를 입력하세요.

---

## Log Format

```
### YYYY-MM-DD | neuron_id | action
- 학습 내용 요약
- 적용한 프로젝트
- 결과/산출물
```

### action types:
- `start` → sparking (학습 시작)
- `progress` → sparking 유지 (진행 중)
- `complete` → fired (습득 완료)
- `risk_resolve` → risk 노드 제거
